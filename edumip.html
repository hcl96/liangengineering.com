Mobile Inverted Pendulum

About eduMIP

The Mobile Inverted Pendulum is a classical controls problem with the added challenge of balancing the pendulum on wheels. As part of a controls graduate class, I develop an optimal state space controller using linear quadratic regulator (LQR) and implemented. In addition to obtaining the control law, a complementary filter is designed to obtain a stable dynamic state. To realize the balancing, a C program was coded to demonstrate the robot in balance.

The eduMIP is an educational Mobile Inverted Pendulum kit developed by Coordinated Robotics Lab at UCSD. It is designed for teaching controls and benchmarking controllers. The robot features a BeagleBone Blue with Linux OS, common IO architectures and even an embedded IMU perfect for dynamical controller implementation
